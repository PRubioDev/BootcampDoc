### Installation

```shell
sudo apt update
sudo apt install software-properties-common
sudo add-apt-repository --yes --update ppa:ansible/ansible
sudo apt install ansible
```

### Ejercicio 2

```yaml
---
# Copiar un fichero .txt al servidor por ejemplo en /tmp/test.txt y a√±adirle la linea "esto es una prueba"
- hosts: all
  tasks:
    - name: Copy file
      ansible.builtin.copy:
        src: test.txt
        dest: /tmp/test.txt
        follow: no
    - name: Add line
      ansible.builtin.lineinfile:
        path: /tmp/test.txt
        line: esto es una prueba
```


```shell
sudo ansible-playbook copy-text.yaml -i inventory
sudo ansible all -i inventory/ -a "cat /tmp/test.txt"
```
